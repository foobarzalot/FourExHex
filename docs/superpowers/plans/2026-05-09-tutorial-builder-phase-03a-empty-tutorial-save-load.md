# TutorialBuilder Phase 3a — Empty-Tutorial Save/Load Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.
>
> **Read first:** `CLAUDE.md` (strict TDD; rebuild-before-launch; manual-test-after-every-change). `ARCHITECTURE.md` (layered view; save layer position). The master plan: `docs/superpowers/plans/2026-05-09-tutorial-builder-master.md`. The design spec: `docs/superpowers/specs/2026-05-09-tutorial-builder-design.md` §"Data model" (`Tutorial` only), §"Save/load", §"JSON v3 schema".

**Goal:** Stand up tutorial save/load end-to-end with an *empty* `Tutorial` POCO (no `Beat` types yet). Bump `SaveSerializer` to v3 with an optional top-level `Tutorial` block, soften the deserializer to accept v2 *and* v3 (for backwards compatibility with existing on-disk saves), wire the topbar's already-built Save Tutorial / Load Tutorial buttons to in-scene dialogs, and verify the round-trip of a painted map draft persists across save/load. No `Beat` / `AnchorRef` / `DismissOn` types — those land in 3b/3c/Phase 7.

**Architecture:** `Tutorial` is a pure C# POCO under `scripts/Tutorial/`. The save format gets one optional new field; the deserializer accepts both versions so existing user saves keep loading. `SaveStore` gets three new methods that delegate to existing private helpers (`ListSlotsIn`, `LoadSlotIn`) parameterized by directory; `WriteTutorial` mirrors `WriteMapSlot`'s shape. `TutorialBuilderScene` copies the dialog pattern verbatim from `MapEditorScene`, with substituted titles and store calls.

**Tech Stack:** Godot 4.6.1 (.NET) + C# (net8.0) + xUnit. `System.Text.Json` over hand-written DTOs (existing pattern). Tutorials live at `user://tutorials/`.

    private void OnLoadDialogInput(InputEvent @event)
    {
        if (@event is not InputEventKey keyEvent || !keyEvent.Pressed || keyEvent.Echo) return;
        if (keyEvent.Keycode != Key.Escape) return;

    private void OpenLoadDialog()
    {
        if (_loadDialog == null || _loadDialogList == null) return;

        foreach (Node child in _loadDialogList.GetChildren())
        {
            child.QueueFree();
        }
        IReadOnlyList<SaveSlotInfo> slots = _saveStore.ListTutorials();
        if (slots.Count == 0)
        {
            var emptyLabel = new Label { Text = "No tutorials found." };
            emptyLabel.AddThemeFontSizeOverride("font_size", 18);
            _loadDialogList.AddChild(emptyLabel);

                Alignment = HorizontalAlignment.Left,

        _loadDialog.PopupCentered();

        }

        _loadErrorDialog.PopupCentered();

    {
        var dt = System.DateTimeOffset.FromUnixTimeSeconds(unixSeconds).LocalDateTime;
        return dt.ToString("yyyy-MM-dd HH:mm");
